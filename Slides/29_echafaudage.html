3 - Construction de l'Échafaudage

Embranchements

Générique

Plat

Orthogonal


- on distingue deux cas particuliers pour les Embranchements
- dans le cas générique on utilise la partiton de sphère
- dans le cas ou les branches sont toutes environ dans un meme plan:
- partition de la sphère en quartier d'orange
- alignement avec les axes: insertion d'un hexa subdvisié avec un échafaudage composé d'un quad par face de l'hexa
-
- nous permettra plus de flexibilité pour une configuration plus adaptée
